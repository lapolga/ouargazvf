L'APK OUARGAZ_CHEF_EQUIPE.apk est généré automatiquement par GitHub Actions.

NE PAS compiler localement. Procédure :
1. Pousser le projet sur GitHub
2. Onglet Actions → workflow "Build OUARGAZ Chef Equipe APK" → Run workflow
3. Télécharger l'artifact OUARGAZ_CHEF_EQUIPE_APK

Voir README_BUILD_APK.md à la racine du dossier projet.
